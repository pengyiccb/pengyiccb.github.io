    centos7x安装MariaDB服务

    centos
    创建时间:2020-04-20 23:50
    阅读:

    • 下载安装包

    • 按顺序安装依赖环境包

    yum install rsync nmap lsof perl-DBI nc
    rpm -ivh jemalloc-3.6.0-1.el7.x86_64.rpm
    rpm -ivh jemalloc-devel-3.6.0-1.el7.x86_64.rpm

    • 卸载冲突的MariaDB-libs

      • 先搜索
        rpm -qa | grep mariadb-libs
      • 后删除
        rpm -ev --nodeps mariadb-libs-5-5.60-1.el7_5.x86_64

    • 安装boost-devel依赖环境
      yum install boost-devel.x86_64

    • 导入MariaDB的key
      rpm --import http://yum.mariadb.org/RPM-GPG-KEY-MariaDB

    • 安装galera环境
      rpm -ivh galera-25.3.19-1.rhel7.el7.centos.x86_64.rpm

    • 安装libaio(此步骤在安装10.4.8时需要)
      wget http://mirror.centos.org/centos/6/os/x86_64/Packages/libaio-0.3.107-10.el6.x86_64.rpm
      rpm -ivh libaio-0.3.107-10.el6.x86_64.rpm

    • 安装MariaDB的四个核心包
      rpm -ivh MariaDB-common-10.4.8-1.el7.centos.x86_64.rpm MariaDB-compat-10.4.8-1.el7.centos.x86_64.rpm MariaDB-client-10.4.8-1.el7.centos.x86_64.rpm galera-4-26.4.2-1.rhel7.el7.centos.x86_64.rpm MariaDB-server-10.4.8-1.el7.centos.x86_64.rpm

    安装玩后还没完，还需要配置数据库，在命令行执行如下代码用以启用MySQL（MariaDB）服务
    service mysql start
    启动成功后运行如下命令进行安全配置
    mysql_secere_installation

    1. 输入当前密码，初次安装后时没有密码的，直接回车
    2. 询问是否使用unix_socket进行身份验证: N
    3. 为root设置密码：y
    4. 输入root的新密码为：root
    5. 确认root输入的密码：root
    6. 是否移出匿名用户，这个随意，建议删除：y
    7. 拒绝用户远程登陆，这个建议开启：y
    8. 删除test库，可以保留：y
    9. 重新加载权限表：y

    当你看到Thanks for using MariaDB!这句话，表明配置成功了

    这个时候还没完，当我们进行远程连接的时候，还需要给用户赋予远程连接的权限

    • 输入密码
      mysql -u root -p
    • 赋予root用户远程连接权限
      grant all privileges on *.* to 'root'@'%' identified by '密码'
      flush privileges
